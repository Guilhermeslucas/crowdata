Crowdata

Easily crowdsource the analysis of your documents

Download this project as a .zip file Download this project as a tar.gz file

CrowData is a tool to collaborate on the verification or release of data that otherwise would be hard or impossible to get via automatic tools.

Where is used?

VozData is a website from La Nacion in Argentina to convert scanned PDF documents from senate spending into an usable data set.
